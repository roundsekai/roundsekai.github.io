使用Stable Diffusion1.0模型,本地生成图片

Created2024-02-05|Updated2024-11-21|Software
|Word Count:1.6k|Reading Time:8mins|Post Views:

简介

SDXL1.0模型是Stability AI 推出的一款免费图像生成模型，输出的图像分辨率提高到了1024×1024，简化了步骤,让文生图变得更为便捷高效,几句话就可以输出美观的图像。
项目地址：https://github.com/Stability-AI/generative-models
模型下载：
base model ：https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0
refiner model：https://huggingface.co/stabilityai/stable-diffusion-xl-refiner-1.0
ComfyUI：https://github.com/comfyanonymous/ComfyUI
Windows Version：https://github.com/comfyanonymous/ComfyUI/releases/download/latest/ComfyUI_windows_portable_nvidia_cu121_or_cpu.7z
在线体验：https://stability.ai
模型大小：6~7GB
推荐配置：6GB或以上显存的NVIDIA显卡,1GB/s或以上的硬盘读写速度。

用法

首先下载ConfyUI(windows用户直接下载整合包)和模型,解压后把模型放入主目录中的ComfyUI➡models➡checkpoints 文件夹中,例如：G:\Project\Stable Diffusion1.0\ComfyUI_windows_portable\ComfyUI\models\checkpoints
英伟达显卡运行run_nvidia_gpu.bat,没有可以运行run_cpu.bat(显存不足自动开启低显存模式),稍等片刻自动打开网页程序


可以看到图形界面有正面提示词和负面提示词,可以运行默认的提示词先看看效果,点击Queue Prompt生成,第一次出图一般都是下面这张：

注意第一次使用要先加载模型,加载速度和硬盘读写速度以及设备性能有关。如果配置较低请耐心等待，并没有卡住,只是比较慢罢了。
在Empty Latent Image 下可以修改生成的图像大小,最大像素为1024×1024，默认512×512
在Ksample下可以修改步数和CFG值(classifier-free guidance scale),并不是越高越好,可自行摸索。
图像生成后自动保存到ComfyUI下的output文件夹中,也可以右键图像,自定义存储位置。
想要生成复杂的图形一定要多加一些提示词,可以自行谷歌相关模板。

视频教程


视频的后半段为stable-diffusion-webui的安装教程,配置要求更高,显存≥12GB推荐试一试,可以安装额外的插件,更为方便。

提示词

能否生成好的图像很大程度上取决于提示词的用法,下面是一些样本可供参考
来源：SDXL 1.0 多种风格高质量prompt

Generic Starter Prompt通用提示词
Prompt ：breathtaking {prompt} . award-winning, professional, highly detailed
Negative ：anime, cartoon, graphic, text, painting, crayon, graphite, abstract glitch, blurry
示例:
Prompt ：breathtaking selfie photograph of astronaut floating in space, earth in the background. award-winning, professional, highly detailed
Negative ：anime, cartoon, graphic, text, painting, crayon, graphite, abstract glitch, blurry

Anime动画风格:

Negative :photo, deformed, black and white, realism, disfigured, low contrast
示例：
Prompt ：anime artwork an empty classroom. anime style, key visual, vibrant, studio anime, highly detailed
Negative ：photo, deformed, black and white, realism, disfigured, low contrast
