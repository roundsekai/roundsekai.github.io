自建VPS操作手册

准备工作

开始之前，先准备好下列项目。
1.一张可以在境外网站付款的信用卡（Visa / Mastercard 均可）没有可以在万能的某宝上搜索目标VPS的礼品卡。
2.一个电子邮箱（Gmail / Outlook / QQ 郵箱均可）用来接受验证码。
3.电脑（Windows 10 或 macOS 10.14 以上）或手机。
4.大约2小时的空闲时间。

💡 协议升級提示
本教程已淘汰旧版的 Outline/Shadowsocks，采用无需购买域名即可「借用」大厂网站进行伪装的Reality技术。安全级别最高，且自带可视化流量流量管理后台。

🛒 某宝替代方案速查
沒有境外信用卡：搜「VPS 礼品卡代充」「Vultr 充值 / DigitalOcean 充值 / Linode 充值」，卖家会直接帮你充到账户，也有卖「预开好的成品号」可以直接登录使用。
沒有海外邮箱：搜「Gmail 成品号 / Outlook 海外号」，通常几块钱一个，拿到后第一时间改密码、绑定自己的手机号二次验证。
⚠️ 提醒：成品号有被原主找回的风险，建议只用作注册VPS类一次性用途，不要拿来关联你的日常身份。

⚠️ 地区选择与IP被墙警告
中国用户推荐选择 美国西海岸（洛杉矶/西雅图）、新加坡或日本大阪（比東京干净）。尽量避开 Vultr 的 日本东京节点——那里是重灾区，开出來的新IP很多直接就是被封锁的。如果第一次开出來的VPSping不通，在 VPS 后台点「重建」换一个IP或新开一台服务器即可（Digital Ocean类的直接重开服务器就行）。
按天/小时计费的推荐多开几个不同位置的服务器分别ping一下，搭起来后测下网速再考虑要不要继续使用。带宽低于50Mbps的基本没有意义。三网速度也各不相同，适合自己的就是最好的。

创建服务器

在你的服务商后台创建一台新的服务器（Droplet/Instance），请务必按照以下配置选择：

选项 选什么 说明
System Image Ubuntu 22.04 LTS 或 24.04 最稳定的Linux版本
Server Size $5~$6/mo（1 CPU, 1GB RAM） VPN用途完全都用
Authentication Password 设置一个包含大小写和数字的复杂密码

记录IP地址和密码
开机完成后（通常需要1分钟），在控制台找到并记录以下两个信息：

IP Address：形如 192.168.xx.xx 的一串数字
Password：你刚才设置的密码（或者系統分配的 root 密码）

成功连接后，你会看到一个欢迎界面，并出现 root@xxx:~# 这样的提示符，这意味着你已经进入了服务器。

设置后台账号密码

安装过程中，屏幕会弹出提示让你进行基础设置（按提示输入）：

是否设置面板账户密码入口? [y/n]: 输入 y 回车
•请设置您的账户名: 输入一个你记得住的名字，比如 admin
•请设置您的密码: 输入你的后台登录密码
•请设置面板访问端口: 输入一个 10000 到 65000 之间的数字，比如 23456
•完成后，系统会提示绿色的「x-ui 面板启动成功」。

有的时候会直接给你初始的账户密码，进入面板后请自行更改

登录网页控制面板

打开你的浏览器，在网址栏输入 https://你的服务器IP:你刚才设置的端口（例如 https://142.250.xx.xx:23456）。输入刚才设置的账号密码或安装x-ui随机生成的账号密码登录。

如果网页打不开，通常是因为服务商的云防火墙（Firewall / Security Group）未开放该端口。请去VPS后台的防火墙设置中放行该端口，或放行所有TCP端口。

创建Reality节点(核心步骤)

　进入面板后，点击左侧菜单的面板设置 (Inbounds)，点击 + 添加入站 (Add Inbound)，按照以下严格填写：
备注 (Remark) 随便填，例如MyReality
协议 (Protocol) 选择 vless
端口 (Port) 填 443（最真实的 HTTPS 端口）
传输配置 (Reality) 将Reality开关打开
目标网站 (Target) 填　www.microsoft.com:443　(这是我们要伪装的对象，也可以换成apple,sony,nvidia等)
Server Names (SNI) 填　www.microsoft.com
密钥 (Keys) 点击旁边的 Get New Cert 自动生成一对密钥

获取一键导入连接

在 Inbounds 列表中，找到你刚才创建的节点，编辑流量信息，单位为GB，在Clients列表中，添加客户，默认留空，流量不限制使用，最后一行添加刚才设置的节点信息。点击刚才设置的Client,会出现二维码，直接用代理软件扫码导入即可或复制连接，你会得到一串以 vless:// 开头的长代码，再导入进代理软件。这就是你的终极「科学上网钥匙🔑」。

在设备上配置客户端

由于 Reality 是最新协议，老旧的客户端无法识别。请务必使用以下指定的现代客户端。

Windows — v2rayN / Clash Verge Rev

推荐 V2rayN：

•前往 GitHub 下载 v2rayN-Core.zip 并解压缩打开。
•确保剪切板里已经复制了你的 vless:// 连接。
•在软件界面按 Ctrl + V 直接导入或左上角通过剪切板导入。
•选中节点，按回车（测试延迟），然后在右下角任务栏目图标右键 -> 系统代理 -> 自动配置系统代理 即可上网，右侧模式自选。
Clash Verge Rev (更美观，支持分流)：前往其 GitHub 下载，左侧点击「代理」，右上角「新建」->「从剪切板导入」。

Android — v2rayNG

•在 Google Play 或 Github 下载 v2rayNG。
•打开 App，点击右上角的 + 号。
•选择 从剪切板导入（或扫描面板Clients的二维码）。
•点击右下角的 V 字圆标启动连接。

iOS — Shadowrocket (小火箭)

⚠️ 需要美区 Apple ID
你需要登录美区的 App Store，购买并下载 Shadowrocket (约 $2.99)也可某宝搜索。
复制 vless:// 链接后，打开 Shadowrocket。
软件会自动识别剪切板内容，弹出提示，点击 添加。
回到首頁，打开顶部的连接开关，完成 VPN 配置授权即可。

VPS测试

去程： https://tools.ipip.net/traceroute.php
去程测试可以查看你的流量都经过了那些服务器最终到达VPS，一般来说中转的越少延迟越低。主要目的是看你的VPS走的是什么线路，有没有走CN2,GIA高速网络。

回程：bash <(curl -Ls https://Check.Place) -N

在本地后台输入ssh root@服务器IP　输入密码登陆后，直接粘贴这一行，选择测试项目后自动运行，可以查看VPS的网络测试报告

硬件： bash <(curl -Ls https://Check.Place) -H

查看VPS的硬件情况，通常对搭建节点无影响，用于推理模型或对硬件质量有要求参考价值会较高。

IP： bash <(curl -Ls https://Check.Place) -I

查看VPS的IP类型，风险因素和流媒体解锁服务

稳定性： https://www.itdog.cn/ping

直接在网页上输入VPS　IP地址，大概15分钟时间，直观的看出三网的稳定性和各地连接状况。
